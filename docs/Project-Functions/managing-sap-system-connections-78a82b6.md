# Managing SAP System Connections

Learn how to save the connection information for a remote system in Visual Studio Code \(VS Code\).



## Connection Manager for SAP Systems

The connection manager for SAP systems enables you to store connection information for remote SAP systems. By saving your connection details, you can authenticate more quickly when working with SAP development tools. For more information, see [https://marketplace.visualstudio.com/items?itemName=SAPOSS.sap-ux-sap-systems-ext](https://marketplace.visualstudio.com/items?itemName=SAPOSS.sap-ux-sap-systems-ext).

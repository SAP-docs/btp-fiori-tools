# Generating Mock Data with AI

Learn how to generate mock data with AI to produce meaningful and contextually relevant data.

You can generate mock data by using the property names of entities to generate meaningful and contextually relevant data with AI.



<a name="loio815c31061dc74a8385bbd7f2d5a24d2c__section_lsy_b1t_51c"/>

## Prerequisites

Ensure the following prerequisites are met:

-   You have a subscription to SAP Build Code or have signed up for [SAP Build Code Test Drive](https://developers.sap.com/mission.sap-build-code-test-drive.html?sap-outbound-id=4E44C2A19D38B160BF5539329FA7ECC83942C1AD). For more information, see [What is SAP Build Code](https://help.sap.com/docs/build_code/d0d8f5bfc3d640478854e6f4e7c7584a/504854f457cc4fbf9f79136dbc773618.html).

-   You have an EDMX project that includes a `metadata.xml` file for the service.



## Procedure

To generate mock data with AI, perform the following steps:

1.  In the *Page Editor*, click the <span class="SAP-icons-TNT-V3">î„š</span> \(*Generate Mock Data with AI*\) icon.
2.  Click *Generate*.

Three records of mock data are generated. Depending on the size, generation may take some time.
